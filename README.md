# UHL-Parser

Group timeseries CSV data from UHL trend system export by timestamp and export to CSV

## Formatting

Converts data in the format:
xxxxxxx			1
xxxxxxx					2
xxxxxxx							3
xxxxxxx									4
yyyyyyy			1
yyyyyyy					2
yyyyyyy							3
yyyyyyy									4

To:

xxxxxxx			1		2		3		4
yyyyyyy			1		2		3		4
